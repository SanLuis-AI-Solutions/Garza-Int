# /ai-solution-architecture - AI Solution Architecture

$ARGUMENTS

---

## Task

Define a complete AI/automation architecture with data flow, tools, and guardrails.

Use skills: `ai-agents-architect`, `agent-tool-builder`, `agent-memory-systems`.

---

## Steps

1. **System Overview**
   - Core capabilities and user journeys

2. **Data & Integrations**
   - Data sources, APIs, tools, and permissions

3. **Agent Design**
   - Roles, responsibilities, memory, and handoff logic

4. **Safety & Reliability**
   - Guardrails, validation, monitoring, fallback paths

---

## Output

- Architecture summary
- Component list and responsibilities
- Data flow and integration map
- Risks, mitigations, and open questions

---

## Usage Examples

```
/ai-solution-architecture AI assistant for customer support + CRM
/ai-solution-architecture automate sales pipeline with AI + n8n
```
