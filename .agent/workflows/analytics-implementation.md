# /analytics-implementation - Analytics Implementation

$ARGUMENTS

---

## Task

Create a complete tracking and analytics plan that proves ROI and guides decisions.

Use skills: `analytics-tracking`.

---

## Steps

1. **KPI Definition**
   - Business and product KPIs, plus baseline metrics

2. **Event Taxonomy**
   - Naming, properties, and tracking rules

3. **Instrumentation Plan**
   - Where events fire and validation steps

4. **Reporting**
   - Dashboard views and cadence

---

## Output

- KPI list and definitions
- Event taxonomy table
- Implementation checklist
- Reporting dashboard outline

---

## Usage Examples

```
/analytics-implementation tracking for SaaS onboarding
/analytics-implementation marketing funnel analytics
```
