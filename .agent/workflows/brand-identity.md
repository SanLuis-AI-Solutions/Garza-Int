# /brand-identity - Brand Identity & Positioning

$ARGUMENTS

---

## Task

Create a clear brand identity system: positioning, voice, visual direction, and messaging hierarchy.

Use skills: `frontend-design`, `ui-ux-pro-max`, `documentation-templates`.

---

## Steps

0. **Load Project Brief (if present)**
   - Read `PROJECT_BRIEF.md` and use it to prefill context
   - Only ask for missing or unclear items

1. **Positioning & Differentiation**
   - Category and competitors
   - Unique value proposition
   - “Only we can say this” statement

2. **Audience & Jobs‑To‑Be‑Done**
   - Primary persona and top 3 pains
   - Desired outcomes and objections

3. **Brand Voice & Tone**
   - 3–5 voice traits
   - Do/Don’t examples
   - Signature phrases/taglines

4. **Messaging Architecture**
   - Core promise
   - 3 supporting pillars
   - Proof points

5. **Visual Direction**
   - Mood/feel keywords
   - Color direction (60‑30‑10)
   - Typography direction
   - UI motifs and imagery style

---

## Output

- Positioning statement
- Messaging pillars + proof points
- Voice & tone guide
- Visual direction summary
- Taglines (3–5)

---

## Usage

```
/brand-identity reposition jewelry repair brand
```
