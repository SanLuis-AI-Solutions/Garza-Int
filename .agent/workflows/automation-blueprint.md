# /automation-blueprint - Automation Blueprint (n8n)

$ARGUMENTS

---

## Task

Translate requirements into an n8n automation plan with nodes, data mapping, and error handling.

Use skills: `n8n-workflow-patterns`, `n8n-node-configuration`, `n8n-validation-expert`, `n8n-expression-syntax`, `n8n-mcp-tools-expert`, `n8n-code-javascript`, `n8n-code-python`.

---

## Steps

1. **Triggers & Inputs**
   - Start events, schedules, and input schemas

2. **Workflow Design**
   - Node sequence, branching, retries, and error paths

3. **Data Mapping**
   - Field mapping, transforms, and validation rules

4. **Ops & Monitoring**
   - Logging, alerts, runbook, and test cases

---

## Output

- Workflow diagram (text)
- Node-by-node configuration list
- Validation and error-handling plan
- Test checklist

---

## Usage Examples

```
/automation-blueprint lead capture -> CRM -> email follow-up
/automation-blueprint invoice ingestion -> accounting sync
```
