# UI Artifacts (Multi-Model Loop)

This folder holds lightweight, repo-committed artifacts for UI work so the team can:
- see what was decided (and why),
- reproduce designs from Stitch/Gemini,
- review changes (Opus critique),
- and verify implementation against a spec.

## Folder Convention
Create one folder per UI effort:

`Docs/artifacts/ui/YYYY-MM-DD--<slug>/`

Required files:
- `00-brief.md`
- `01-gemini-fast-options.md`
- `02-stitch-links.md`
- `03-opus-review.md`
- `04-implementation-notes.md`

Optional:
- `screenshot.webp` (before/after)
- `source.html` (only if it contains no secrets; prefer links/IDs instead)
