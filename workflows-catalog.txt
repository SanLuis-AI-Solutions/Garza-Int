- /ui-artifact-loop
  Description: Multi-model UI loop (Codex leads; Gemini Fast explores; Gemini Pro + Stitch converges; Opus reviews) with repo-committed artifacts.
  Uses skills: (not specified)
  Steps:
    - 1. Create `Docs/artifacts/ui/YYYY-MM-DD--<slug>/` and write `00-brief.md`
    - 2. Generate 3-7 UI directions with Gemini Fast (`01-gemini-fast-options.md`)
    - 3. Choose 1 direction (Codex) and record the decision in `00-brief.md`
    - 4. Generate Stitch screens + save IDs/URLs (`02-stitch-links.md`)
    - 5. Opus critique with change requests (`03-opus-review.md`)
    - 6. Implement, verify (`npm run build`), and document (`04-implementation-notes.md`)
  Outputs:
    - Artifact folder with 00-04 files (brief, options, Stitch links, Opus review, implementation notes)
    - Verification notes (build + smoke checks)
